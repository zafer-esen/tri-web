TriCera v0.4

TriCera is a model checker for C programs, developed at Uppsala University.

TriCera accepts programs in a C-like language, including the keywords:

  • assert(condition) — verify that a condition holds at this point in the program.
  • assume(condition) — assume that a condition holds.
  • atomic { ... } — execute a block of statements atomically.
  • within (cond) { ... } — execute a block atomically before cond expires (timed automata).
  • thread <name> { ... } — declare a singleton thread.
  • thread[<id>] <name> { ... } — declare an infinitely replicated thread.
  • clock, duration — declare clocks or time period variables.
  • chan <name> — declare a binary communication channel, used via chan_send and chan_receive.

For more information, see the GitHub page.
